## 实现内容

### 1. 后端API接口 (courtroom_web.py)

#### 新增接口：
- `POST /api/avatar/upload` - 上传用户头像
  - 支持格式：PNG, JPG, JPEG, GIF, WebP
  - 文件大小限制：5MB
  - 参数：avatar (文件), user_id (用户ID)
  
- `GET /api/avatar/<filename>` - 获取用户头像
  - 返回图片文件流
  
- `DELETE /api/avatar/<filename>` - 删除用户头像
  - 需要确认操作

#### 安全特性：
- 文件类型白名单验证
- 文件大小限制检查
- 路径遍历攻击防护

## 使用方法

### 运行Web应用：
```bash
python courtroom_web.py
```

访问 http://localhost:5000，点击"👤 用户设置"标签页

### 运行测试：
```bash
# 安装测试依赖
pip install -r requirements.txt

# 运行所有测试
pytest test_avatar.py -v

# 运行特定测试类
pytest test_avatar.py::TestAvatarUpload -v

# 查看测试覆盖率
pytest test_avatar.py --cov=courtroom_web --cov-report=html
```
